What is ChatChannels?

ChatChannels is an easy-to-use Spigot plugin that allows server owners to create a multitude of chatting channels with simple yaml files. ChatChannels also features integration with the Factions plugin and the Feudal plugin, allowing chat channels to be made for factions/kingdoms.
